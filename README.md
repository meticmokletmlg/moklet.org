<h2>🏃‍♂️ How to run</h2>

Clone this repository:

```bash
git clone https://github.com/mokletdev/moklet.org.git
```

Install dependencies and setup husky

```bash
npm install
npm run prepare
```

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
